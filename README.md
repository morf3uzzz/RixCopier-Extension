# RixAI Копир

Умное расширение для браузера, созданное специально для экономии времени (и токенов!) при сборе данных для нейросетей (LLM).

## Что умеет:
- **Чистый парсинг:** вытягивает чистый контент с любого сайта (обрезает меню, рекламу и футеры).
- **Выбор мышкой:** позволяет "накликать" нужные блоки текста (или зажать `Alt/Option` для мульти-выбора).
- **Три формата буфера:** идеальный Markdown (таблицы/заголовки), обычный текст или сухой Clean-режим (для экономии токенов).
- **Мульти-сессия:** склеивает контент с разных вкладок в один файл для массовой выгрузки.
- **Встроенный калькулятор:** сразу показывает примерный вес контента и всей сессии в токенах LLM.
- **Авто-атрибуция:** автоматически пришивает к тексту: откуда взято (ссылка), дату и автора.

## Установка
1. Скачайте этот репозиторий.
2. В браузере (Chrome / Edge / Яндекс) откройте страницу расширений (`chrome://extensions/`).
3. Включите **Режим разработчика** в правом верхнем углу.
4. Нажмите **Загрузить распакованное расширение** и выберите папку с исходным кодом.
